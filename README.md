# Game_Snake 🐍
***Простая реализация классической игры «Змейка» на языке программирования Python с использованием модуля `turtle`. Игра создана для отдыха и расслабления: она не требует сложных решений, помогает отвлечься после рабочего дня и вызывает ностальгию по играм детства.***

# Управление
"W" -- *движение вверх*

"S" -- *движение вниз*

"A" -- *движение влево*

"D" -- *движение вправо*

# Правила игры
1. Управляйте змейкой, чтобы собрать еду 🍎🐛
2. После каждого съеденного яблока змейка начинает увеличиваться
3. Цель - собрать как можно больше очков!
‼️Обратите внимание, что соприкосновение со стеной или своим телом заканчивается поражением ☠️
